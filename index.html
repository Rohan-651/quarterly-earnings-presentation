Financial Formula — Return on Equity (ROE)

ROE is defined as:

\(\displaystyle \text{ROE} = \dfrac{\text{Net Income}}{\text{Average Shareholders' Equity}}\)

If net income = \(N\) and equity = \(E\), then \(\text{ROE} = N/E\).
